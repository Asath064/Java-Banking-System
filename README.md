# Simple Banking Application

## Description

A Java console-based Banking Application that allows users to perform basic banking operations such as depositing money, withdrawing funds, and checking account balances.

The project is developed using Object-Oriented Programming (OOP) principles and demonstrates real-world banking system functionality with proper validation and error handling.

## Features

- Deposit Money
- Withdraw Money
- Check Account Balance
- View Account Details
- Error Handling for Invalid Transactions
- Insufficient Balance Validation
- Menu-Driven Interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Conditional Statements
- Loops
- Exception Handling Concepts

## How to Run

Compile:

```bash
javac BankAccount.java BankingApplication.java
```

Run:

```bash
java BankingApplication
```
